# Axiant Referral Landing Baseline

Starter project for a referral-source landing page flow.

## Included pages

- `index.html` — general referral partner page
- `declined-deals.html` — page for brokers, lenders, MCA shops, and others with declined files
- `referral-form.html` — simple intake form page
- `styles.css` — shared styling
- `script.js` — placeholder JS behavior

## Goal

This baseline is designed to feel:

- professional
- calm
- credible
- relationship-driven
- not overly salesy

## Suggested next steps in Cursor

1. Replace placeholder branding, logo, and colors.
2. Connect the referral form to your CRM or form tool.
3. Turn the static pages into Astro, Next.js, or React components.
4. Add real trust signals:
   - lender count
   - partner categories
   - testimonials
   - turnaround expectations
   - compliance/disclaimer language
5. Add more SEO support:
   - FAQ schema
   - internal links
   - partner-specific pages
   - location/service pages if relevant

## Suggested prompt for Cursor

Convert this static baseline into a modern component-based site with reusable sections for hero, referral source cards, trust section, FAQ, and intake form. Keep the tone professional and relationship-first.
